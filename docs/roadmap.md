# Roadmap

## Near term

- add support for selecting vector layers by name
- add GeoParquet read/write helpers
- add validation for duplicate features and null-heavy fields
- add better mixed-geometry handling for shapefile export
- add multiprocessing for temporal composites

## Medium term

- add cloud optimized GeoTIFF export options
- add STAC-oriented helpers
- add raster stack alignment checks
- add optional xarray support for composite workflows

## Long term

- add plugin interface for project-specific QA rules
- add remote catalog connectors
- add standardized processing manifests for reproducibility
